# E-commerce-Data-Analytics-Project-Python-SQL-
This project involves analyzing a large dataset from a Brazilian e-commerce company using SQL for data extraction and Python (Pandas, Seaborn) for data manipulation and visualization.
Goal: To extract actionable insights regarding sales performance, customer behavior, and seller analytics.

# Key Features / Questions Answered

Data Loading: Using Python to dump CSV files into a MySQL database.
Basic Analysis: Finding unique customer locations and counting orders per month.
Intermediate Analysis: Calculating total revenue per category and analyzing payment methods.
Advanced Analysis: Calculating retention rates and year-over-year growth
